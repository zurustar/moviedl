# moviedl

YouTube など 1000 以上のサイトから動画をダウンロードできる GUI アプリです。

## 特徴

- **シングルバイナリ** — インストール不要。ダウンロードしてそのまま使えます
- **Mac / Windows 対応** — macOS (Apple Silicon) と Windows (64bit)
- **高画質ダウンロード** — ffmpeg 同梱済みのリリース版は映像・音声を別々に取得して結合するため、最高画質で保存できます
- **yt-dlp 自動インストール** — 初回起動時に案内に従ってインストールするだけで使えます
- **ダウンロードキュー** — URL を連続して登録でき、順番に処理されます
- **同時ダウンロード数の設定** — 同時に実行するダウンロード数を 1〜10 の範囲でプルダウンから選べます（既定は 1）
- **進捗の詳細表示** — ダウンロード中は進捗率・ファイルサイズ・速度・残り時間・経過時間を表示
- **キャンセル** — 待機中・ダウンロード中のアイテムをいつでもキャンセルできます

## ダウンロード

**Windows**: [Releases](../../releases) ページから `moviedl-windows.zip` を入手してください。

**macOS**: リリースバイナリはコード署名されていないため、Gatekeeper にマルウェアと判定されて実行できない場合があります。**ローカルビルドを推奨します**（後述の「ビルド方法」を参照）。

## 使い方

1. アプリを起動する
2. 初回のみ「yt-dlp をインストール」ボタンをクリック
3. 動画の URL を入力して Enter または「ダウンロード」ボタンを押す（複数登録可）
4. ダウンロードは登録順に処理されます。待機中のアイテムは「待機中 1/3」のように表示されます
5. 「同時ダウンロード数」プルダウンで、同時に実行する件数を 1〜10 の範囲で選べます（既定は 1）。数を増やすと待機中のアイテムが自動的に並行ダウンロードされます
6. ダウンロード中は進捗率・ファイルサイズ・速度・残り時間・経過時間が確認できます
7. キャンセルしたい場合は各アイテムの「×」ボタンを押してください
8. デフォルトの保存先は `~/Downloads`。「変更」ボタンで変更できます

## ビルド方法

Go 1.22 以上と [Wails v2](https://wails.io) が必要です。macOS ユーザーはこちらでビルドしてください。

```bash
# 1. Wails のインストール（初回のみ）
make install-wails

# 2. ビルド → build/bin/moviedl
make build
```

その他のターゲット:

```bash
make build-universal  # macOS ユニバーサルバイナリ（arm64 + amd64）
make dev              # 開発モード（ホットリロード付き）
make test             # テスト実行
make help             # ターゲット一覧
```

詳細なビルド手順・CI・リポジトリ構成は [CONTRIBUTING.md](CONTRIBUTING.md) を参照してください。
