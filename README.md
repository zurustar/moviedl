# moviedl

YouTube など 1000 以上のサイトから動画をダウンロードできる GUI アプリです。

## 特徴

- **シングルバイナリ** — インストール不要。ダウンロードしてそのまま使えます
- **Mac / Windows 対応** — macOS (Apple Silicon) と Windows (64bit)
- **高画質ダウンロード** — ffmpeg 同梱済みのリリース版は映像・音声を別々に取得して結合するため、最高画質で保存できます
- **yt-dlp 自動インストール** — 初回起動時に案内に従ってインストールするだけで使えます
- **ダウンロードキュー** — URL を連続して登録でき、1件ずつ順番に処理されます
- **進捗の詳細表示** — ダウンロード中は進捗率・ファイルサイズ・速度・残り時間・経過時間を表示
- **キャンセル** — 待機中・ダウンロード中のアイテムをいつでもキャンセルできます

## ダウンロード

[Releases](../../releases) ページから最新版を入手してください。

| プラットフォーム | ファイル |
|---|---|
| macOS (Apple Silicon / Intel) | `moviedl-macos.zip` |
| Windows 64bit | `moviedl-windows.zip` |

### macOS の注意

初回起動時に「開発元を確認できない」と表示される場合は、Finder で右クリック →「開く」を選択してください。

## 使い方

1. アプリを起動する
2. 初回のみ「yt-dlp をインストール」ボタンをクリック
3. 動画の URL を入力して Enter または「ダウンロード」ボタンを押す（複数登録可）
4. ダウンロードは1件ずつ順番に処理されます。待機中のアイテムは「待機中 1/3」のように表示されます
5. ダウンロード中は進捗率・ファイルサイズ・速度・残り時間・経過時間が確認できます
6. キャンセルしたい場合は各アイテムの「×」ボタンを押してください
7. デフォルトの保存先は `~/Downloads`。「変更」ボタンで変更できます

## ビルド方法

Go 1.22 以上と [Wails v2](https://wails.io) が必要です。

```bash
go install github.com/wailsapp/wails/v2/cmd/wails@latest
wails build
```

リリース用ユニバーサルバイナリ（macOS）:

```bash
wails build -platform darwin/universal
```

> リリースビルドは GitHub Actions が自動で行います。`v*` タグを push するだけで Mac / Windows 向けバイナリが作成されます。
