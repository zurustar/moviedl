# moviedl 要件定義

## 概要

YouTube など多数の動画配信サイトから動画をダウンロードできる GUI アプリケーション。

---

## 機能要件

### ダウンロード

- 動画サイトの URL を入力して動画をダウンロードできる
- 対応サイトは 1000 以上
- 最高画質でダウンロードできる
- 保存先ディレクトリをユーザーが選択できる（デフォルトは `~/Downloads`）

### ダウンロード状態とリスト管理

アイテムは以下の3つのリストのいずれかに属する：

| リスト | 説明 |
|---|---|
| **ダウンロード待ちキュー** | 登録済みで未開始のアイテム |
| **ダウンロード実行中リスト** | 現在ダウンロードが進行中のアイテム |
| **一時停止中リスト** | 一時停止されたアイテム |

### キュー

- 複数の URL を連続して登録できる
- URL の登録と実際のダウンロード処理は独立している
- **URL 登録時の動画情報取得（yt-dlp による問い合わせ）は非同期に行い、取得完了を待たずに次の URL を入力・登録できる**
- 取得中の URL は UI 上に「取得中」アイテムとして可視化する
- **アプリウィンドウ内のどこにブラウザタブやテキストから URL をドラッグ&ドロップしても、「追加」ボタンを押さずに即座に登録処理を開始する**（ドロップ可能範囲は入力欄に限定せずウィンドウ全体）
- ドロップ時にアプリ外のページ遷移・別ウィンドウ表示を起こさない
- ドロップされたデータに複数の URL が含まれる場合（改行区切り）、それぞれを個別に登録する
- **同時ダウンロード数（自動補充の上限）をユーザーが 1〜10 の範囲でプルダウンメニューから選択できる**（デフォルトは 1）
- **実行中リストのアイテム数が設定値を下回ると、待ちキューにアイテムがある限り先頭から順に実行中リストへ移動し、設定値まで補充する**
- 実行中リストのアイテム数が設定値に達している場合は自動補充は行われない
- 設定値を引き上げたときは、待ちキューに余裕分のアイテムがあれば即座に補充される
- 手動開始（後述）による並行ダウンロードは上限の対象外（設定値を超えて開始できる）

### 一時停止・再開

- 実行中リストのアイテムを個別に一時停止できる（実行中 → 一時停止中へ移動）
- 一時停止によって実行中リストが空になった場合は、自動補充ルールにより待ちキューから次のアイテムが開始される
- 一時停止中リストのアイテムはユーザーが手動で再開できる（一時停止中 → 実行中へ移動）

### 手動開始

- 待ちキューおよび一時停止中リストのアイテムは、ユーザーが手動で実行中リストへ移動できる
- これにより実行中リストに複数アイテムを並行ダウンロードさせることができる（同時ダウンロード数の上限なし）

### ファイル選択

- プレイリストや複数動画を含む URL を登録した場合、含まれる動画の一覧を表示する
- ユーザーはその一覧から個別にダウンロードするファイルを選択できる
- 選択されたファイルはそれぞれ独立したキューアイテムとして待ちキューに追加される

### キャンセル

- 3 つのリストのいずれに属するアイテムもいつでもキャンセル・削除できる

### エラー時の挙動

- ダウンロードがエラー終了したアイテムは **リストから自動削除してはならない**
- エラー状態のアイテムはエラーであることが視覚的にわかる形でリストに残し続け、エラー理由を表示する
- ユーザーは以下の操作ができる:
  - **リトライ**: 同じ URL でダウンロードを再試行する（待ちキューに戻り、自動補充で実行される）
  - **削除**: エラーアイテムをリストから明示的に取り除く
  - **URL コピー**: そのアイテムの元 URL をクリップボードにコピーする（リトライしても直らない場合に、通常のブラウザ等で代替ダウンロードするため）
- 理由: エラー原因はディスクフル・一時的なネットワーク障害など、ユーザーの対処でリトライ可能なケースが多い。自動消去するとユーザーは原因を確認できず、復旧手段も失う
- 後処理（ffmpeg 結合）の失敗のように**リトライでは直らない**場合もあるため、URL コピーで他手段に逃がせるようにする

### URL のコピー

- ダウンロードリストの各アイテムから、登録時の元 URL をワンクリックでクリップボードにコピーできる（エラー時に限らず、全状態で可能）

### 自動削除（成功・キャンセル時のみ）

- ダウンロードが正常完了したアイテムはリストから自動削除する
- ユーザーがキャンセルしたアイテムはリストから自動削除する
- **エラー終了したアイテムは自動削除の対象外**（上記「エラー時の挙動」を参照）

### 進捗表示

ダウンロード中は以下を表示する：

- 進捗率（%）
- ファイルサイズ
- ダウンロード速度
- 残り時間（ETA）
- 経過時間

### バージョン表示

- アプリ画面上でビルドのバージョンを確認できる（リリースビルドは `v0.1.8` などのタグ、それ以外は `dev` とビルド日を表示）

### 依存ツール管理

- 必要なツールが未インストールの場合、アプリ内から案内またはインストールできる
- yt-dlp は全プラットフォームでアプリ内からダウンロード・配置する
- ffmpeg は Windows ではアプリ内からダウンロード・配置する（バイナリには同梱しない）。macOS は Homebrew のインストールを案内する

---

## 非機能要件

### 配布形態

- インストール不要、1 ファイルで動作する
- macOS (Apple Silicon / Intel) および Windows 64bit に対応

### ファイル管理

- ダウンロード完了後のファイルは保存先フォルダに置かれる
- 保存先に同名ファイルが既に存在する場合は `タイトル (1).mp4` のように連番を付ける
- ダウンロード中に生成される中間ファイルは、完了後に確実に削除される
- 中間ファイルが残存した場合にユーザーが気づいて手動削除できること

### 自動リリース

- バージョンタグの push で macOS・Windows 向けバイナリが自動ビルドされ公開される
